# 任务规划: PINN-YODEL 项目

## 目标声明
火箭推进剂屈服应力预测系统 (PINN-YODEL) - 基于物理信息神经网络的混合建模项目

**主要目标**:
1. 维护和优化现有 PINN-YODEL 模型
2. 提升模型预测精度和泛化能力
3. 完善文档和代码质量

**成功标准**:
- 训练 Loss < 20 Pa
- 测试 R² > 0.8
- 物理参数预测合理 (Φ_m ∈ [0.72, 0.74], G_max ≈ 2.2×10⁵)
- 代码模块化、可维护

### Phase 2: 多保真度学习实施 (Multi-Fidelity Learning) ⚡ 当前阶段
**状态**: in_progress
**目标**: 实现低保真度 → 高保真度的两阶段训练策略

**背景**:
- 用户需求: 使用合成数据训练低保真度模型,用论文数据微调高保真度模型
- 数据来源: 论文 "Constitutive Modeling of Rheological Behavior of Cement Paste" (2025)
- 策略: 冻结网络参数 + 物理层微调

**子阶段 2.1: 论文数据提取与预处理** ✅
**状态**: complete
**任务**:
- [ ] 从论文 1 提取实验数据 (Table 3, 5, 6)
  - Table 3: 纯水泥浆料 (5 种配方)
  - Table 5: 含 SCMs 浆料 (60 种配方)
  - Table 6: 含增塑剂浆料 (16 种配方)
- [ ] 参数映射: φ → Phi, d50 → d50, PSD → sigma
- [ ] 数据筛选: 选择 φ ∈ [0.35, 0.44], τ₀ ∈ [50, 200] Pa
- [ ] 创建高保真度数据集: `data/high_fidelity/cement_paste_data.csv`
- [ ] 数据统计分析: 分布、相关性、物理合理性

**预期输出**:
- `data/high_fidelity/cement_paste_data.csv` (~30-40 样本)
- `data/high_fidelity/data_analysis.png` (数据分布图)
- `docs/论文数据提取报告.md`
