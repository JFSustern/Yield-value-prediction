## 架构发现

### 1. 核心架构
```
输入 [5维] → 神经网络 [3×128] → 物理参数 [Φ_m, G_max] → YODEL 机理 → τ₀
```

**关键设计特点**:
- **混合建模**: 神经网络预测隐变量,物理层执行确定性计算
- **物理约束**: m₁ 强制计算而非预测,确保 m₁ ∝ d50⁻² 的物理规律
- **Loss 函数**: Log-MSE 适配长尾分布 (0~3500 Pa)

### 2. 模块结构

## 技术发现

### 1. YODEL 物理机理

**核心公式**:
```
τ₀ = m₁ * [Φ(Φ - Φ_c)²] / [Φ_m(Φ_m - Φ)]
```

**关键参数**:
- **Φ_c** (渗透阈值): 颗粒网络形成的临界固含量,由 d50 和 sigma 决定
- **Φ_m** (最大堆积): 受混合功 Emix 动力学修正
- **m₁** (几何预因子): 与颗粒间作用力相关,满足 m₁ ∝ d50⁻²
- **G_max**: 颗粒接触刚度,基准值 ≈ 80000

**物理约束**:
- Φ < Φ_m (避免堵塞)
- Φ_c < Φ < Φ_m (网络形成但未饱和)
- m₁ > 0, G_max > 0

### 2. 工序模拟 (9阶段)

**关键时间点**:
1. **T1 (48 min)**: 混合中期,Phi=Phi_1
2. **T_new (83 min)**: 高速混合后,固化剂未加,Phi=Phi_1
3. **T2 (111 min)**: 固化剂加入并混合后,Phi=Phi_2

**固化剂效应**:
- 比例: 4-6% (ratio_curing ∈ [0.04, 0.06])
- 对 Phi 的影响: Phi_2 = Phi_1 + ratio_curing
- 典型范围: Phi_2 ∈ [0.63, 0.67]

### 3. 数据生成策略

**参数范围**:
- d50 ∈ [20, 32] μm
- sigma ∈ [1.4, 1.9]
- Phi_2 ∈ [0.63, 0.67]
- ratio_curing ∈ [0.04, 0.06]

**物理过滤条件**:
- Tau0_1 ∈ [70, 120] Pa
- Tau0_2 < 200 Pa
- Phi < Phi_m

**数据规模**:
- 生成样本: ~15000
- 过滤后: ~2700 有效样本
- 训练/测试分割: 80/20

### 用户需求

### 论文 1: Constitutive Modeling of Rheological Behavior of Cement Paste (2025)

**基本信息**:
- 期刊: Materials (MDPI)
- 材料: 水泥浆料 (Portland Cement + SCMs: 粉煤灰/矿渣/石灰石粉)
- 预测目标: 屈服应力 (τ₀) 和塑性粘度 (η)

**数据特点**:
- **实验数据量**: 65 种配方
- **固含量范围**: φ ∈ [0.28, 0.44] (对应 w/b = 0.30-0.50)
- **屈服应力范围**: 0-300 Pa (纯水泥), 最高达 5463 Pa (SF 掺合料)
- **物理参数**:
  - 粒径分布 (PSD): d50 = 2.94-47.15 μm
  - 比表面积 (SSA): 2800-150,000 cm²/g
  - 密度: 2.23-3.15 g/cm³

**核心模型**:
- **YODEL 模型**: τ = m₁ × φ³ / [φₘₐₓ(φₘₐₓ - φ)]
- **K-D 方程**: μ = μ₀(1 - φ/φₘ)^(-[η]φₘ)
- **拟合精度**: R² > 0.98 (屈服应力), R² > 0.85 (塑性粘度)

**可用性评估**: ⭐⭐⭐⭐⭐ **强烈推荐**
- ✅ **数据完整**: 提供完整的 PSD、SSA、密度、化学组成
- ✅ **参数丰富**: 65 种配方,涵盖多种 SCMs 和固含量
- ✅ **模型兼容**: 使用 YODEL 模型,与项目完全一致
- ✅ **表格清晰**: Table 3, 5, 6 提供详细实验数据
- ⚠️ **物理差异**: 水泥浆料 vs 火箭推进剂 (颗粒化学性质不同)

### 论文 2: The Yield Stress of Concentrated Flocculated Suspensions (1999)

**基本信息**:
- 期刊: Journal of Rheology
- 材料: Al₂O₃ 氧化铝颗粒悬浮液
- 预测目标: 屈服应力 (τ_ymax)

**数据特点**:
- **实验数据量**: 4 种粒径 (AKP-15, 20, 30, 50)
- **固含量范围**: φ ∈ [0.1, 0.6]
- **屈服应力范围**: 0-7000 Pa
- **物理参数**:
  - 粒径分布: d₅₀ = 0.332-0.548 μm, d₉₀ = 0.945-0.391 μm
  - BET 比表面积: 3.8-10.5 m²/g
  - 密度: 4.0 g/cm³

**核心模型**:
- **张力强度模型**: τ_ymax = AΦC_N(Φ) / (24πh₀²d)
- **无量纲屈服应力**: τ_ymax/T_ymax = f(Φ, d/h₀)
- **粒径依赖**: τ_ymax ∝ d⁻²

**可用性评估**: ⭐⭐⭐ **可用但需要适配**
- ✅ **理论深入**: 详细分析网络结构对屈服应力的影响
- ✅ **粒径效应**: 研究了粒径分布对屈服应力的影响
- ⚠️ **数据量少**: 仅 4 种粒径,样本量有限
- ⚠️ **模型不同**: 使用张力强度模型,非 YODEL
- ⚠️ **材料差异**: 氧化铝颗粒 vs 火箭推进剂

**理由**:
1. **数据量最大**: 65 种配方,足够支持微调
2. **模型完全兼容**: 使用 YODEL 模型,无需修改物理层
3. **参数完整**: 提供 PSD、SSA、密度、化学组成
4. **屈服应力范围合理**: 0-300 Pa (部分配方与项目接近)

### 数据融合策略

#### 阶段 1: 低保真度训练 (已完成)
- 使用合成数据 (~2700 样本)
- 训练 PINN 模型,Loss ~16 Pa, R² > 0.8

#### 阶段 2: 高保真度微调 (待实施)
**数据预处理**:
1. **提取论文 1 数据**:
   - Table 3: 纯水泥浆料 (5 种配方)
   - Table 5: 含 SCMs 浆料 (60 种配方)
   - Table 6: 含增塑剂浆料 (16 种配方)

2. **参数映射**:
   ```
   论文参数          →  项目参数
   ----------------------------------------
   φ (固含量)        →  Phi
   d50 (中位粒径)    →  d50
   σ (几何标准差)    →  sigma (需从 PSD 计算)
   w/b (水胶比)      →  通过 φ 反推
   ```

3. **数据筛选**:
   - 选择 φ ∈ [0.35, 0.44] 的配方 (接近项目范围)
   - 选择 τ₀ ∈ [50, 200] Pa 的配方 (接近项目目标)

**微调策略**:
1. **冻结网络参数**: 冻结 PINN 的神经网络层
2. **微调物理层**: 仅微调 YODEL 物理层的参数 (m₁, Φ_m, G_max)
3. **Loss 函数**: 保持 Log-MSE
4. **训练轮数**: 50-100 epochs (避免过拟合)

**预期效果**:
- 提升模型在真实数据上的泛化能力
- 验证 YODEL 物理机理的普适性
- 为后续获取真实推进剂数据做准备

### 物理差异性考虑

**水泥浆料 vs 火箭推进剂**:
| 特性 | 水泥浆料 | 火箭推进剂 |
|------|----------|------------|
| 颗粒类型 | 水泥、粉煤灰、矿渣 | 氧化剂颗粒 (AP, Al) |
| 化学反应 | 水化反应 | 固化反应 |
| 粒径范围 | 1-100 μm | 20-32 μm (项目) |
| 固含量 | 0.28-0.44 | 0.63-0.67 (项目) |
| 屈服应力 | 0-300 Pa | 70-120 Pa (项目) |

**迁移学习合理性**:
- ✅ **物理机理相似**: 都是固液两相流,YODEL 机理普适
- ✅ **参数可映射**: PSD、固含量、粒径等参数通用
- ⚠️ **化学性质不同**: 水化 vs 固化,需要注意
- ⚠️ **固含量偏低**: 论文数据 φ < 0.44,项目 φ > 0.63
