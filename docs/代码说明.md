# 代码结构与使用说明

## 1. 项目概述

### 1.1 项目目标
本项目是一个**物理信息神经网络(Physics-Informed Neural Network, PINN)**系统，用于预测**火箭固体推进剂浆料的屈服应力(Tau0)**。核心创新是将**YODEL物理机理模型**（基于颗粒网络理论）嵌入深度学习框架，实现"数据驱动+物理约束"的混合建模方法，既保证预测精度，又保证物理意义。

### 1.2 核心技术
- **物理引擎**: YODEL屈服应力模型（Flatt & Bowen, 2006）
- **神经网络**: 物理信息神经网络(PINN)架构
- **数据生成**: 工序模拟的合成数据生成器
- **优化方法**: Adam优化器 + Log-MSE损失函数 + 梯度裁剪
- **部署平台**: PyTorch + MPS/CUDA/CPU支持

### 1.3 代码总览
- **核心模块**: 5个（物理、模型、数据、训练、分析）

## 3. 核心模块详解

### 3.1 物理引擎 (src/physics/yodel.py)

#### 3.1.1 模块概述
YODEL是一个颗粒网络理论模型，用于预测悬浊液(Suspension)的屈服应力。该模块实现了YODEL的四个关键公式。

#### 3.1.2 关键函数

**1. `calc_phi_c(d50, sigma)` - 计算渗透阈值**
```python
def calc_phi_c(d50, sigma):
    """
    计算渗透阈值 Phi_c (Percolation Threshold)

    文献来源: Flatt & Bowen (2006)
    原始公式: Phi_c = 0.28 * (1 + CV)
    其中 CV = sqrt(exp((ln(sigma))^2) - 1)

    Args:
        d50: 中位径 (Tensor) [um]
        sigma: 几何标准差 (Tensor) [无量纲]
    Returns:
        Phi_c (Tensor) [无量纲]
    """
```

**物理含义**:
- Phi_c是颗粒开始形成网络的临界固含量
- 低于Phi_c时，物料为牛顿流体（无屈服值）
- 高于Phi_c时，颗粒间形成接触网络，产生屈服应力
- sigma越大（颗粒分布越宽），Phi_c越大

**实现细节**:
```python
ln_sigma = torch.log(sigma)
cv = torch.sqrt(torch.exp(ln_sigma**2) - 1)  # 几何标准差→变异系数
phi_c = 0.28 * (1 + cv)
```

**2. `calc_m1(d50, G_max)` - 计算几何预因子**
```python
def calc_m1(d50, G_max):
    """
    计算几何预因子 m1

    文献公式 (Eq. 41): m1 = (1.8/π⁴) * (G_max / R²)

    Args:
        d50: 粒径 (Tensor) [um]
        G_max: 颗粒间最大作用力 (Tensor) [Force]
    Returns:
        m1 (Tensor) [Pa]
    """
```

**物理含义**:
- m1是YODEL方程的强度因子，反映颗粒间相互作用的强度
- 与粒径的平方呈反比（细颗粒→强度高）
- 与颗粒间最大作用力成正比

**实现细节**:
```python
const = 1.8 / (π^4)  # 几何常数
R = d50 / 2.0         # 粒径转半径
return const * G_max / (R^2 + 1e-8)  # +1e-8防止除零
```

**3. `calc_phi_m_dynamic(phi_m0, Emix, k_E, phi_m_ultimate)` - 动力学修正**
```python
def calc_phi_m_dynamic(phi_m0, Emix, k_E=1e-8, phi_m_ultimate=0.74):
    """
    混合功对最大堆积密度的动力学修正

    公式: Phi_m = Phi_m0 + (Phi_ultimate - Phi_m0) * (1 - exp(-k * Emix))

    Args:
        phi_m0: 初始最大堆积 [无量纲]
        Emix: 混合功 [J]
        k_E: 速率常数 [1/J]
        phi_m_ultimate: 极限堆积 (FCC=0.74)
    Returns:
        Phi_m_eff (Tensor) [无量纲]
    """
```

**物理含义**:
- 混合功使颗粒重新排列，提高堆积效率
- 堆积密度从初值phi_m0逐步增加到极限值(FCC=0.74)
- k_E=1e-8反映能量强度（E~10^8 J时的渐进演化）

**4. `yodel_mechanism(phi, phi_m, phi_c, m1)` - YODEL主方程**
```python
def yodel_mechanism(phi, phi_m, phi_c, m1):
    """
    YODEL 主方程

    核心公式: tau0 = m1 * [Φ(Φ - Φ_c)²] / [Φ_m(Φ_m - Φ)]

    Args:
        phi: 固含量 [无量纲]
        phi_m: 最大堆积 [无量纲]
        phi_c: 渗透阈值 [无量纲]
        m1: 强度因子 [Pa]
    Returns:
        tau0 (Tensor) [Pa]
    """
```

**公式推导**:
```
分子: Φ × (Φ - Φ_c)²
  - 表示颗粒接触网络强度
  - Φ越高、(Φ - Φ_c)越大 → 屈服值越大

分母: Φ_m × (Φ_m - Φ)
  - (Φ_m - Φ)表示堆积空间
  - 当Φ→Φ_m时，分母→0，τ₀→∞（堵塞效应）

约束条件:
  - Φ > Φ_c: 颗粒已形成网络
  - Φ < Φ_m: 未完全堵塞
  - 当Φ < Φ_c时，mask=0，τ₀=0（无屈服值）
```

**数值稳定性处理**:
```python
epsilon = 1e-6
diff_c = torch.relu(diff_c)  # 确保非负
diff_m = torch.relu(diff_m) + epsilon  # 分母保证正数
mask = (phi > phi_c).float()  # 物理约束掩码
```

#### 3.1.3 公式实现总结

| 函数 | 输入维度 | 输出维度 | 物理意义 |
|------|---------|---------|--------|
| calc_phi_c | [d50, sigma] | Phi_c | 渗透阈值 |
| calc_m1 | [d50, G_max] | m1 | 强度因子 |
| calc_phi_m_dynamic | [phi_m0, Emix] | Phi_m_eff | 有效堆积 |
| yodel_mechanism | [phi, phi_m, phi_c, m1] | Tau0 | 屈服应力 |

### 3.2 模型定义 (src/model/)

#### 3.2.1 PINN模型架构 (src/model/pinn.py)

**整体架构**:
```
输入层 [5维]
  ├─ Phi(固含量)
  ├─ d50(中位径)
  ├─ sigma(粒径分布)
  ├─ Emix(混合功)
  └─ Temp(温度)
     ↓
[输入预处理] 归一化
  Scale = [1.0, 0.03, 0.6, 3.0e-8, 0.02]
     ↓
[神经网络层] 参数预测器
  ├─ Linear(5, 128)
  ├─ Tanh
  ├─ Linear(128, 128)
  ├─ Tanh
  ├─ Linear(128, 128)
  ├─ Tanh
  └─ Linear(128, 2) → [Phi_m_raw, G_max_raw]
     ↓
[物理约束转换] 解码为物理量
  ├─ Phi_m_pred = Phi + Sigmoid(Phi_m_raw) × (0.76 - Phi)
  └─ G_max_pred = Softplus(G_max_raw) × 80000
     ↓
[物理层] 确定性计算
  ├─ m1 = calc_m1(d50, G_max_pred)
  ├─ Phi_c = calc_phi_c(d50, sigma)
  └─ Tau0 = yodel_mechanism(Phi, Phi_m_pred, Phi_c, m1)
     ↓
输出: [Tau0_pred, (Phi_m_pred, m1_pred, G_max_pred)]
```

**关键代码**:
```python
class YodelPINN(nn.Module):
    def __init__(self, input_dim=5, hidden_dim=64):
        super().__init__()

        # 参数预测神经网络
        self.net = nn.Sequential(
            nn.Linear(input_dim, hidden_dim),
            nn.Tanh(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.Tanh(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.Tanh(),
            nn.Linear(hidden_dim, 2)  # [Phi_m_raw, G_max_raw]
        )

        # G_max基准值
        self.register_buffer('g_max_base', torch.tensor(80000.0))

    def forward(self, x):
        # 输入: [batch, 5]
        # 1. 归一化
        scale = torch.tensor([1.0, 0.03, 0.6, 3.0e-8, 0.02], device=x.device)
        x_norm = x * scale

        # 2. NN预测
        out = self.net(x_norm)  # [batch, 2]

        # 3. 物理量解码
        phi = x[:, 0]
        d50 = x[:, 1]
        sigma = x[:, 2]

        # Phi_m: Sigmoid约束到[Phi, 0.76]
        phi_m_pred = phi + torch.sigmoid(out[:, 0]) * (0.76 - phi)

        # G_max: Softplus确保>0
        g_max_pred = torch.nn.functional.softplus(out[:, 1]) * self.g_max_base

        # 4. 物理计算
        m1_pred = calc_m1(d50, g_max_pred)
        phi_c = calc_phi_c(d50, sigma)

        # 5. YODEL求解
        tau0_pred = yodel_mechanism(phi, phi_m_pred, phi_c, m1_pred)

        return tau0_pred, (phi_m_pred, m1_pred, g_max_pred)
```

**设计亮点**:
1. **m1被强制计算**: m1不由NN预测，而是用物理公式计算，保证物理规律
2. **Phi_m约束**: Sigmoid + 缩放确保Phi_m > Phi且< 0.76
3. **G_max约束**: Softplus确保G_max > 0
4. **可微物理层**: 所有物理计算都可微，支持反向传播

#### 3.2.2 基线模型 (src/model/baseline.py)

**模型架构**:
```python
class PureNN(nn.Module):
    def __init__(self, input_dim=5, hidden_dim=128):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(input_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, 1)  # 直接输出Tau0
        )

    def forward(self, x):
        # 输入归一化（与PINN保持一致）
        scale = torch.tensor([1.0, 0.03, 0.6, 3.0e-8, 0.02], device=x.device)
        x_norm = x * scale

        # 直接预测
        out = self.net(x_norm)

        # 确保非负
        return torch.nn.functional.softplus(out)
```

**特点**:
- 纯数据驱动，无物理约束
- 用于与PINN进行性能对比
- 验证物理约束的作用

### 3.4 训练与评估 (src/)

#### 3.4.1 训练脚本 (src/train.py)

**训练流程**:
```python
def train():
    # 1. 数据准备
    train_df = pd.read_csv("data/synthetic/train_data.csv")
    X = train_df[feature_cols].values  # [batch, 5]
    y = train_df[target_col].values    # [batch]

    dataset = TensorDataset(X_tensor, y_tensor)

    optimizer = optim.Adam(model.parameters(), lr=1e-3)
    criterion = nn.MSELoss()

    # 3. 训练循环
    for epoch in range(200):
        for batch_X, batch_y in loader:
            # Forward pass
            tau0_pred, params = model(batch_X)

            # 计算损失 (Log-MSE处理长尾分布)
            loss = torch.log(criterion(tau0_pred, batch_y) + 1.0)

            # Backward pass
            optimizer.zero_grad()
            loss.backward()
            torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)
            optimizer.step()

    # 4. 保存模型
    torch.save(model.state_dict(), "models/yodel_pinn.pth")
```

**关键配置**:

| 配置项 | 值 | 说明 |
|-------|-----|-----|
| 优化器 | Adam | lr=1e-3 |
| 损失函数 | Log-MSE | torch.log(MSE + 1.0) |
| 批大小 | 64 | Batch size |
| 轮数 | 200 | Epochs |
| 梯度裁剪 | max_norm=1.0 | 防止爆炸梯度 |
| 隐层维度 | 128 | 3层×128 |
| 输入维度 | 5 | [Phi, d50, sigma, Emix, Temp] |

**超参数调优指南**:
- **learning_rate**: 当Loss不下降时，降低至5e-4；收敛缓慢时升高至2e-3
- **hidden_dim**: 小数据集用64，大数据集用128-256
- **epochs**: 观察validation loss，早停策略
- **batch_size**: GPU受限时降低；数据充足时可提高

### 7.2 注释风格

**函数文档**:
```python
def calc_phi_c(d50, sigma):
    """
    计算渗透阈值 Phi_c

    文献: Flatt & Bowen (2006)
    公式: Phi_c = 0.28 * (1 + CV)

    Args:
        d50 (Tensor): 中位径 [um]
        sigma (Tensor): 几何标准差 [-]

    Returns:
        Tensor: 渗透阈值Phi_c [-]

    Example:
        >>> d50 = torch.tensor(25.0)
        >>> sigma = torch.tensor(1.5)
        >>> phi_c = calc_phi_c(d50, sigma)
    """
```

**代码注释**:
```python
# 标量计算: 从几何标准差转换到变异系数
ln_sigma = torch.log(sigma)
cv = torch.sqrt(torch.exp(ln_sigma**2) - 1)

# 代入YODEL公式
phi_c = 0.28 * (1 + cv)
```

## 8. 常见问题

### 8.1 训练不收敛

**症状**: Loss不下降，训练曲线波动大

**原因和解决方案**:

1. **学习率过高**
   - 症状: Loss剧烈波动
   - 解决: 降低lr至5e-4或1e-4
   ```python
   optimizer = optim.Adam(model.parameters(), lr=5e-4)
   ```

2. **学习率过低**
   - 症状: Loss下降缓慢
   - 解决: 升高lr至2e-3或5e-3
   ```python
   optimizer = optim.Adam(model.parameters(), lr=2e-3)
   ```

3. **数据质量问题**
   - 症状: 训练集和测试集Loss差异大
   - 解决: 检查数据分布，运行特征分析
   ```bash
   python src/analysis/feature_analysis.py
   ```

4. **网络过深**
   - 症状: 梯度消失
   - 解决: 减少层数或使用ReLU替代Tanh
   ```python
   nn.Linear(128, 128),
   nn.ReLU(),  # 替代Tanh
   ```
