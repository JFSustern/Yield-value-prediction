### 2.4 混合功计算方法

**公式**：
```
Power (W) = 2π × (Speed / 60) × Torque
Emix (J) = Σ Power × Δt

其中：
- Speed: 转速 (rpm)
- Torque: 扭矩 (N·m)
- Δt: 时间间隔 (秒，通常为60秒)
```

**实现代码**（src/data/loader.py）：
```python
def load_excel_data(data_dir="data/20251121处理后/"):
```

## 3. 合成数据生成

### 3.1 生成原理

**为什么需要合成数据**：
- 真实数据样本数量有限（仅17个工序），无法满足深度学习的数据量需求
- 真实数据分布集中，覆盖的参数空间有限
- 需要覆盖更广泛的PSD参数组合

**半合成策略**：
1. 使用真实工序流程（扭矩/转速/温度/时间阶段）
2. 随机采样粒径分布参数（d50, sigma）
3. 通过物理机理（YODEL模型）计算对应的屈服应力

### 3.4 物理计算流程

**YODEL机理应用**：

YODEL模型是一个基于物理的屈服应力模型，核心方程为：

```
τ₀ = m₁ × Phi(Phi - Φc)² / [Φₘ(Φₘ - Phi)]
```

其中：
- **τ₀**：屈服应力 (Pa)
- **Phi**：固含量
- **Φc**：渗透阈值 (Percolation Threshold)
- **Φₘ**：最大堆积密度 (Maximum Packing Fraction)
- **m₁**：强度因子 (Strength Prefactor, Pa)

**计算步骤**（以T1时间点为例）：

**Step 1: 计算渗透阈值 Φc**
```python
def calc_phi_c(d50, sigma):
    """
    Phi_c = 0.28 * (1 + CV)
    其中 CV = sqrt(exp((ln(sigma))^2) - 1)
    """
    ln_sigma = torch.log(sigma)
    cv = torch.sqrt(torch.exp(ln_sigma**2) - 1)
    phi_c = 0.28 * (1 + cv)
    return phi_c
```

**Step 2: 计算强度因子 m1**
```python
def calc_m1(d50, G_max):
    """
    m₁ = (1.8/π⁴) × (G_max / R_v50²) × F_sigma
    简化为：m₁ = (1.8/π⁴) × G_max / R²
    其中 R = d50 / 2
    """
    const = 1.8 / (np.pi ** 4)  # ≈ 0.00183
    R = d50 / 2.0
    m1 = const * G_max / (R ** 2)
    return m1

# G_max的温度修正
g_max_base = 80000.0  # 基础颗粒间作用力
g_max_1 = g_max_base * (1 + 0.05 * (temp_1 - 25.0))
```

**Step 3: 计算最大堆积密度 Φₘ（动力学修正）**
```python
def calc_phi_m_dynamic(phi_m0, Emix, k_E=1e-8, phi_m_ultimate=0.74):
    """
    动力学模型：混合功改善最大堆积
    Φₘ = Φₘ₀ + ΔΦ(1 - exp(-k × E))
    其中：
    - Φₘ₀：初始最大堆积 = 0.65 + 0.1(σ - 1.2)
    - ΔΦ = Φ_FCC - Φₘ₀ ≈ 0.74 - Φₘ₀
    - k_E = 1e-8 /J（高能混合过程参数）
    """
    phi_m0 = 0.65 + 0.1 * (sigma - 1.2)
    delta = 0.74 - phi_m0
    phi_m = phi_m0 + delta * (1 - torch.exp(-k_E * Emix))
    return phi_m
```

**Step 4: 应用YODEL主方程**
```python
def yodel_mechanism(phi, phi_m, phi_c, m1):
    """
    τ₀ = m₁ × Phi(Phi - Φc)² / [Φₘ(Φₘ - Phi)]
    """
    diff_c = phi - phi_c
    diff_c = torch.relu(diff_c)  # 约束Phi ≥ Φc
    numerator = phi * (diff_c ** 2)

    diff_m = phi_m - phi
    diff_m = torch.relu(diff_m) + 1e-6  # 约束Phi < Φₘ
    denominator = phi_m * diff_m

    tau0 = m1 * numerator / denominator

    # 如果Phi < Φc，则τ₀ = 0（流体无屈服应力）
    mask = (phi > phi_c).float()
    return tau0 * mask
```

**计算流程图**：
```
d50, sigma → Phi_c (渗透阈值)
d50, T → m1 (强度因子，含温度修正)
sigma, Emix → Phi_m (最大堆积，含混合功动力学修正)
Phi, Phi_m, Phi_c, m1 → Tau0 (YODEL方程)
```

```python
def validate_physics_constraints(df):
    """验证物理约束"""
    issues = []

    # 检查1：固含量范围
    if (df['Phi(固含量)'] < 0).any() or (df['Phi(固含量)'] > 1).any():
        issues.append("Phi超出[0,1]范围")

    # 检查2：固含量 < 最大堆积
    if (df['Phi(固含量)'] >= df['Phi_m_true(最大堆积)']).any():
        issues.append("Phi >= Phi_m（应满足Phi < Phi_m）")

    # 检查3：屈服应力范围
    if (df['Tau0(屈服应力_Pa)'] < 0).any():
        issues.append("Tau0为负值")

    # 检查4：关键约束（本项目）
    tau0_1_invalid = ~((df['Tau0_1(屈服应力_Pa)'] >= 50) &
                       (df['Tau0_1(屈服应力_Pa)'] <= 150))
    if tau0_1_invalid.any():
        issues.append(f"Tau0_1超出[50,150]范围: {tau0_1_invalid.sum()}个样本")

    return issues

# 运行验证
issues = validate_physics_constraints(df_wide)
if issues:
    print("发现问题:")
    for issue in issues:
        print(f"  - {issue}")
else:
    print("所有样本通过物理约束检查")
```

```python
def detect_anomalies(df, threshold=3):
    """使用Z-score检测异常值"""
    from scipy import stats

    anomalies = {}

    for col in df.select_dtypes(include=['float64', 'int64']).columns:
        z_scores = np.abs(stats.zscore(df[col]))
        anomaly_indices = np.where(z_scores > threshold)[0]

        if len(anomaly_indices) > 0:
            anomalies[col] = {
                'count': len(anomaly_indices),
                'indices': anomaly_indices.tolist(),
                'values': df.iloc[anomaly_indices][col].tolist()
            }

    return anomalies

# 检测异常

for col, info in anomalies.items():
```
