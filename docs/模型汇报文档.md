# 火箭推进剂屈服应力预测系统 - 技术汇报文档

## 项目概述

本项目开发了一个基于**物理信息神经网络(PINN)**的火箭固体推进剂浆料**屈服应力(τ₀)**预测系统。核心创新是将**YODEL物理机理模型**嵌入深度学习框架,实现"数据驱动+物理约束"的混合建模。

---

## 一、数据生成策略

### 1.1 数据生成方法

本系统采用**基于工序模拟的合成数据生成**方法,模拟真实混合工序的9个阶段,追踪能量变化和物理参数演化。

#### 工序流程图

```
初始混合 → 添加氧化剂1 → 添加氧化剂2 → 添加氧化剂3 → 混合结束
(10分钟)   (10分钟)     (10分钟)      (10分钟)     (8分钟)
                                                  ↓
                                           [检查点T1: 48分钟]
                                                  ↓
                                               高速混合
                                              (35分钟)
                                                  ↓
                                           [检查点T_new: 83分钟]
                                              (固化剂未加)
                                                  ↓
                                              添加固化剂
                                                (3分钟)
                                                  ↓
                                               最终混合
                                               (25分钟)
                                                  ↓
                                           [检查点T2: 111分钟]
                                                  ↓
                                                 静置
                                               (15分钟)
```

### 1.2 输入参数范围

#### 基础物性参数 (颗粒粒度分布, PSD)

| 参数 | 符号 | 取值范围 | 单位 | 物理意义 |
|------|------|----------|------|----------|
| 中位径 | d50 | [20.0, 32.0] | μm | 颗粒粒径分布的中位数 |
| 几何标准差 | σ | [1.4, 1.9] | - | 粒径分布的离散程度 |

**采样方式**: 均匀分布随机采样

**物理解释**:
- d50越小,颗粒比表面积越大,颗粒间作用力越强
- σ越大,粒径分布越宽,渗透阈值Φc越高

---

#### 固含量参数

| 参数 | 符号 | 取值范围 | 单位 | 物理意义 |
|------|------|----------|------|----------|
| T2时刻固含量 | Φ₂ | [0.63, 0.67] | - | 固化剂加入后的最终固含量 |
| 固化剂比例 | ratio | [0.015, 0.025] | - | 固化剂占总质量的比例(1.5%-2.5%) |

**T1时刻固含量计算** (固化剂未加):
```
Φ₁ = Φ₂ / [Φ₂ + (1 - ratio) × (1 - Φ₂)]
```

**采样方式**: 均匀分布随机采样

**物理解释**: 固化剂的加入会稀释浆料,降低固含量,因此Φ₁ > Φ₂

---

#### 工序参数 (9个阶段)

每个阶段包含以下参数:

| 阶段 | 时长(min) | 转速(rpm) | 扭矩(Nm) | 温度(°C) |
|------|-----------|-----------|----------|----------|
| 1. 初始混合 | 10 | [22, 26] | [400, 500] | [42, 44] |
| 2. 添加氧化剂1 | 10 | [22, 26] | [450, 550] | [43.5, 44.5] |
| 3. 添加氧化剂2 | 10 | [22, 26] | [600, 700] | [47, 48] |
| 4. 添加氧化剂3 | 10 | [22, 26] | [900, 1000] | [47, 49] |
| 5. 混合结束 | 8 | [22, 26] | [450, 500] | [47, 50] |
| 6. 高速混合 | 35 | [28, 32] | [1950, 2050] | [52, 53] |
| 7. 添加固化剂 | 3 | [10, 14] | [1750, 1850] | [50, 51] |
| 8. 最终混合 | 25 | [28, 32] | [1300, 1500] | [50, 51] |
| 9. 静置 | 15 | 0 | 0 | 25 |

**采样方式**: 每个阶段的转速、扭矩、温度在给定范围内均匀随机采样

**混合功计算**:
```
E_stage = 2π × speed × torque × duration
E_mix(t) = Σ E_stage  (累积至时间t)
```

**关键时间点的混合功统计**:
- E_mix(T1=48min) ≈ 4.0-4.7 × 10⁶ J
- E_mix(T_new=83min) ≈ 1.6-1.9 × 10⁷ J
- E_mix(T2=111min) ≈ 2.3-2.6 × 10⁷ J

---

### 1.3 目标值计算 (YODEL物理模型)

#### YODEL主方程

```
τ₀ = m₁ × [Φ × (Φ - Φc)²] / [Φm × (Φm - Φ)]
```

**物理参数计算**:

1. **渗透阈值 Φc** (Percolation Threshold):
   ```
   CV = √[exp(ln²σ) - 1]  # 变异系数
   Φc = 0.28 × (1 + CV)
   ```
   - 物理意义: 颗粒网络形成的临界固含量
   - 典型值: 0.35-0.45

2. **最大堆积密度 Φm** (Maximum Packing Fraction):
   ```
   Φm0 = 0.65 + 0.1 × (σ - 1.2)  # 初始值
   Φm = Φm0 + (0.74 - Φm0) × [1 - exp(-1×10⁻⁸ × Emix)]  # 动力学修正
   ```
   - 物理意义: 颗粒能达到的最密堆积
   - 典型值: 0.68-0.74

3. **几何预因子 m₁**:
   ```
   m₁ = (1.8/π⁴) × Gmax / (d50/2)²
   Gmax = 80000 × [1 + 0.05 × (T - 25)]  # 温度修正
   ```
   - 物理意义: 颗粒间最大相互作用力
   - 典型值: m₁ ~ 10⁴-10⁶ Pa

## 二、模型架构

### 2.1 PINN混合架构

```
输入层 [5维]
  ↓
归一化层
  ↓
神经网络 (黑盒)
  - 隐藏层1: 128维 + Tanh
  - 隐藏层2: 128维 + Tanh
  - 隐藏层3: 128维 + Tanh
  - 输出层: 2维 [Φm_raw, Gmax_raw]
  ↓
物理解码层 (白盒)
  - Φm = Φ + Sigmoid(Φm_raw) × (0.76 - Φ)  # 确保 Φm ∈ [Φ, 0.76]
  - Gmax = Softplus(Gmax_raw) × 80000      # 确保 Gmax > 0
  ↓
物理计算层 (确定性)
  - Φc = calc_phi_c(d50, σ)               # 渗透阈值
  - m₁ = calc_m1(d50, Gmax)               # 强制计算!
  ↓
YODEL机理层
  - τ₀ = yodel_mechanism(Φ, Φm, Φc, m₁)
  ↓
输出 [1维]: τ₀预测值
```

**关键设计**:
1. **混合建模**: 神经网络预测难以测量的隐变量(Φm, Gmax),物理层计算确定性关系
2. **物理约束**: Sigmoid和Softplus确保预测参数在物理合理范围
3. **强制计算**: m₁通过物理公式计算而非预测,保证 m₁ ∝ d50⁻² 的物理规律

### 2.2 模型参数

| 参数 | 值 |
|------|-----|
| 输入维度 | 5 |

| 隐藏层数 | 3 |
| 激活函数 | Tanh |
| 输出维度 | 2 (Φm, Gmax) |
| 总参数量 | ~50K |

## 三、训练配置

### 3.1 训练超参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 训练轮数 | 200 epochs | - |
| 批量大小 | 64 | - |
| 学习率 | 1×10⁻³ | Adam优化器 |
| 损失函数 | Log-MSE | 适配长尾分布 |
| 梯度裁剪 | 1.0 | 防止梯度爆炸 |
| 设备 | MPS | Apple Silicon GPU加速 |

### 3.2 损失函数

**Log-MSE Loss** (对数均方误差):

```python
Loss = mean[(log(1 + τ₀_pred) - log(1 + τ₀_true))²]
```

**优势**:
- 适配长尾分布 (τ₀范围: 0-200 Pa)
- 对小值更敏感,避免大值主导
- 数值稳定性好
